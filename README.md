### What I learned

```css
When resizing the screen my p elements would move out of position within the payment box, I fixed it using some flexbox properties to keep it in place.
.para-2{
    align-self: center;
    margin-top: -2rem;
    margin-left: 1rem;
    font-family: 'Red Hat Display-Bold';
    color: hsl(223, 47%, 23%);
}

.para-3{
    align-self: flex-end;
    margin-bottom: 1rem;
    margin-left: -5.6rem;
    font-family: 'Red Hat Display';
    color: hsl(223, 47%, 23%);
}

.change-link{
    align-self: center;
    margin-left: 4rem;
    font-family: 'Red Hat Display-Bold';
    font-size: 14px;
    color: hsl(245, 75%, 52%);
}
```
